# Project 2: Simple Linear Regression Model

## Objective

Build a Linear Regression model to predict house prices.

## Technologies Used

- Python
- Pandas
- Scikit-learn

## Results

### Mean Squared Error (MSE)

24.29

### R-Squared Score (R²)

0.67

## Key Insights

- Number of rooms positively influences house prices.
- LSTAT negatively influences house prices.
- The model explains approximately 67% of the variation in housing prices.
